# Font Files

## Garamond Classico Small Caps

Drop your Garamond Classico Small Caps font files here.

### Required Files

Place your TrueType font file here:

- `GaramondClassicoSC.ttf` (TrueType format)

### Where to Get the Font

Purchase from:
- [MyFonts](https://www.myfonts.com/products/garamondclassico-sc-garamond-classico-370803)
- [Linotype](https://www.myfonts.com/collections/garamond-classico-font-linotype/)

### After Adding Files

Once you've added the font files to this directory, the project is already configured to use them. The font will be automatically loaded and applied to the BusinessCard component.

### File Naming

If your font files have different names, update the `src` path in `/app/layout.tsx` to match your actual filenames.
